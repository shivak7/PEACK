This is the data and python code for the paper: Estimating proprioceptive position sense of upper limbs using joint distance and joint angle symmetries
The "Code" folder contains all code used for generating data and figures in this paper.
The "Data" folder holds the extracted kinematic data for the healthy adults (Healthy) and children with CP (CP).
The "Figures" folder will hold some of the figures generated for this paper.


Run "$python Generate_ViconData_Analysis.py" to extract proprioception measures from the kinematic data.
Then run "$python GraphResults.py" to generate the figures used in this paper.
